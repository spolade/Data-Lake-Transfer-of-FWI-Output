# Transferring FWI Output to the Data Lake

This repository contains scripts for transferring Fire Weather Index (FWI) output data generated by ClimateDT simulations to the EUMETSAT Data Lake. The repository is part of the Wildfire FWI use case of the Climate Adaptation Digital Twin (ClimateDT), developed within the framework of the Destination Earth (DestinE) initiative of the European Commission, with ECMWF serving as one of the Entrusted Entities.
